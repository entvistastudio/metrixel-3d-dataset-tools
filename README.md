# Mesh Visualization Tools for PyTorch 3D Datasets

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Python](https://img.shields.io/badge/Python-3.x-blue)

Utilities for visualizing and inspecting 3D mesh datasets stored as **PyTorch `.pt` tensors**.

These tools allow researchers and developers to preview mesh geometry, animate mesh sequences, and debug dataset pipelines used in:

- machine learning
- computer vision
- robotics
- simulation pipelines

Part of the **Metrixel ecosystem** for generating 3D datasets for AI pipelines.

---

## Who This Is For

• PyTorch3D experiments  
• 3D computer vision datasets  
• Robotics / SLAM pipelines  
• Generative 3D model training  
• Debugging mesh dataset pipelines  

---

## Features

• Load PyTorch `.pt` mesh datasets  
• Visualize mesh geometry in 3D  
• Animate mesh sequences with adjustable FPS  
• Inspect vertices, faces, and mesh structure  
• Gracefully handle missing frames  

---

Install dependencies:

pip install -r requirements.txt

Dependencies:

• torch

## Dataset Structure

Expected dataset layout:

generated/
└── meshes/
    └── 000001/
        ├── vert_000_000_000_0000.pt
        ├── face_000_000_000_0000.pt
        ├── vert_000_000_000_0001.pt
        ├── face_000_000_000_0001.pt

---

## Quick Start

Single frame:

python visualize_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame 000_000_000_0000

Animation:

python animate_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame_begin 000_000_000_0000 --frame_end 000_000_000_0134 --fps 30
