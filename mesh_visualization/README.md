# Metrixel Mesh Visualization Tools

This directory contains Python tools for visualizing 3D mesh files (`.pt`) generated by Metrixel. These tools are designed to work seamlessly with Metrixel's coordinate system and output format.

## 🎯 Overview

Metrixel generates 3D mesh sequences as `.pt` files (PyTorch tensors) containing vertices and faces. These visualization tools allow you to:

- **View individual mesh frames** in 3D
- **Animate mesh sequences** with customizable frame rates
- **Use Metrixel coordinate system** (X=Right, Y=Up, Z=Toward)
- **Handle missing frames** gracefully

## 📋 Requirements

### Python Environment
```bash
# Install required packages
pip install torch matplotlib numpy
```

### Metrixel Data Structure
Your Metrixel output should follow this structure:
```
your_project/
├── generated/
│   └── meshes/
│       └── 000001/                    # Sequence ID
│           ├── vert_000_000_000_0000.pt  # Vertices tensor
│           ├── face_000_000_000_0000.pt  # Faces tensor
│           ├── vert_000_000_000_0001.pt
│           ├── face_000_000_000_0001.pt
│           └── ...
```

## 🚀 Quick Start

### 1. Single Frame Visualization
```bash
# View a single mesh frame (--mesh_dataset_root = folder containing sequence IDs, e.g. .../generated/meshes)
python visualize_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame 000_000_000_0000
```

### 2. Animation Sequence
```bash
# Animate frames 0-134 at 30 FPS
python animate_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame_begin 000_000_000_0000 --frame_end 000_000_000_0134 --fps 30

# Animate frames 0-13 at 15 FPS (slower)
python animate_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame_begin 000_000_000_0000 --frame_end 000_000_000_0013 --fps 15
```

## 🎨 Features

### Metrixel Coordinate System
- **X → Right** (horizontal movement)
- **Y → Up** (vertical movement)
- **Z → Toward** (depth toward camera/viewer)
- **Front View**: Camera positioned to look along Z-axis

### Animation Features
- ✅ **Smooth playback** with customizable FPS
- ✅ **Missing frame handling** (skips unavailable frames)
- ✅ **Continuous looping** until window is closed
- ✅ **Real-time progress** tracking in window title
- ✅ **Consistent scaling** across all frames
- ✅ **Light blue mesh** with black edges for clarity

### Error Handling
- ⚠️ **Missing files**: Gracefully skips unavailable frames
- ⚠️ **Invalid sequences**: Clear error messages
- ⚠️ **Corrupted data**: Safe fallback behavior

## 🔧 Configuration

### Mesh dataset path
The tools take the **mesh dataset root** (the folder that contains sequence IDs like `000001`) as a command-line parameter. No environment variables or config files are required:

```bash
# Point to the folder that contains your sequence directories (e.g. .../generated/meshes)
python visualize_mesh_metrixel.py --mesh_dataset_root /path/to/generated/meshes --seq 000001 --frame 000_000_000_0000
```

Expected layout: `{mesh_dataset_root}/{seq}/vert_*.pt` and `face_*.pt` (e.g. if your dataset root is `/data/my_project`, use `--mesh_dataset_root /data/my_project/generated/meshes`).

### Frame Naming Convention
Metrixel uses this naming pattern:
- **Vertices**: `vert_000_000_000_XXXX.pt`
- **Faces**: `face_000_000_000_XXXX.pt`
- **XXXX**: 4-digit frame number (0000, 0001, 0002, etc.)

## 🎬 Animation Controls

## 🐛 Troubleshooting

### Common Issues

#### "No valid mesh frames found"
- Check that `.pt` files exist in the sequence directory
- Verify frame naming convention matches Metrixel output
- Ensure PyTorch can load the tensor files

#### "Missing files for frame"
- Normal behavior - Metrixel may skip certain frames
- Animation will continue with available frames
- Check Metrixel export settings

#### Import Errors
```bash
# Install missing dependencies
pip install torch matplotlib numpy
```

### Performance Tips
- **Lower FPS** (15-20) for smoother playback on slower systems
- **Shorter sequences** for initial testing
- **Close other applications** for better performance
